# Roadmap

## 0.1 Static Web MVP

- Local folder/file diagnosis
- Launch candidate scoring
- Engine and commercial/private file-structure clue detection
- Archive and disc image warnings
- Error text analyzer
- Markdown report export

## 0.2 Better Diagnosis Recipes

- Community-editable JSON engine rules
- Expand engine fingerprints
- Improve commercial/self-developed engine startup-route diagnostics
- Add confidence explanations for each rule
- Add common error dictionary

## 0.3 Asset Explorer

- Safe file manifest view
- Open-format image/audio preview
- Ren'Py project structure helper
- Creator-owned project export report

## 0.4 Windows Desktop

- Real folder picker with full paths
- Package and disc-image metadata preflight without extraction or mounting
- One-click launch for trusted scanned `.exe/.com` entries
- Launch history
- Create shortcuts
- Portable Windows release workflow
- Create launch profiles
- Optional Locale Emulator command template
- Optional Wine/Proton notes for advanced users
- Launch-failure follow-up flow

## 1.0 Community Release

- Signed release builds
- GitHub Pages demo
- Beginner documentation
- Issue templates for engine detection requests
- Privacy and safety review
